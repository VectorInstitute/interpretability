## Interpretable ML using Glassbox ML methods ##

This directory contains example notebooks and explanations demonstrating 11 Glass box models. The methods are grouped in 2 folders:
- [Imaging](https://github.com/VectorInstitute/interpretability-bootcamp/tree/main/reference_implementations/Intepretable-models/Imaging) folder contains implementations for imaging modality.
- [Tabular](https://github.com/VectorInstitute/interpretability-bootcamp/tree/main/reference_implementations/Intepretable-models/Tabular) folder contains implementations for tabular data.
